# OMNEX

A modern, minimal portfolio website for OMNEX - a global provider of Oil & Gas Solutions and Electrical Supplies.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Design Principles

- Minimal, calm, and structured
- Strong typographic hierarchy
- High contrast but neutral palette
- Generous whitespace
- Smooth, subtle motion
